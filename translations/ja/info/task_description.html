これは、 CheckiO でどのようにミッションを解き、それをどのように生かすのかを説明する目的で存在する、導入のミッションです。このミッションを解き終えると、あなたはより複雑なミッションがあるもう一つのステーションへアクセスできるようになります。

そのため、このミッションは最も簡単な内容になっています。入力として二つの数値を受け取り、それらの二数を掛け算した結果を返す関数を書きましょう。

入力: 二つの引数が与えられます。両者は共に整数です。

出力: 整数を返してください。

例: 

{{init_code_tmpl}}

どのように動作するのか:

あなたが問題を解き始めた時、最初のコードはいつも「空っぽ」の関数 (この関数の中身を、提出する回答として埋めていきます) と、その下にある検証用のコードとで構成されています。気を付けるべきことは、関数は値を返すべきなのであって、それを画面に出力すべきではないということです。 {% if interpreter.slug == "js-node" %} つまり、console.log 関数を使うのではなく、return 文を使う必要があります。 {% else %} つまり、print 関数を使うのではなく、return 文を使う必要があります。この Stack Overflow の解説 (英語)を参照してください。 {% endif %}

あなたの関数の後にある検証用のコードは、「Run Code」ボタン を押すことで、まずセルフチェックを行うために利用できます。さらに、 CheckiO はいくつかの追加のテストをシステム内部に持っていて、「Check Solution」ボタン を押すことで、あなたの関数がそれらのテストに合格できるかを確認します。

このミッションを解くには、「空っぽ」の関数を以下のようなコードで置き換える必要があります。

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

改めて「Check Solution」ボタンをクリックしてみてください。

もし回答がすべてのテストに合格すると、右側のパネルへ、祝福のメッセージが以下のようなアクションの提案と共に表示されます (そう、解いただけで話が終わってしまうわけでは無いのです) 。

あなた自身の回答を公開する前に、まず他のプレイヤーの回答へと目を通すことを推奨します。

最後に大事なことですが、いくつかのミッションは末尾に解くためのヒントを持っている場合があります。ただ、このミッションでは既に解き方を説明してしまいましたから、ここでは CheckiO に関するいくつかの興味深い事実をヒントとして掲載することにします。
