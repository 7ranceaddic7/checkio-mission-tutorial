Gdy zaczynasz rozwiązywać misję, kod początkowo zawiera “pustą” funkcję (którą musisz uzupełnić swoim rozwiązaniem), a poniżej znajdują się asserty. Powinieneś zwrócić uwagę na to, że twoja funkcja powinna zwracać wartości, a nie je wyświetlać. {% if interpreter.slug == "js-node" %} W tym celu używaj komendy return zamiast funkcji console.log. {% else %} W tym celu używaj komendy return zamiast funkcji print. Dowiedz się więcej w tym krótkim wyjaśnieniu. {% endif %}

Możesz użyć assertów znajdujących się poniżej twojej funkcji w celu sprawdzenia swojego rozwiązania wciskając przycisk “Run Code”. CheckiO używa też kilku dodatkowych testów aby sprawdzić twoje rozwiązanie po naciśnięciu przycisku “Check Solution”.

Aby rozwiązać zadanie, “pusta” funkcja musi zostać zastąpiona następującym kodem:

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Spróbuj teraz nacisnąć przycisk “Check Solution”.

Jeśli rozwiązanie przejdzie wszystkie testy, na prawym panelu powinny pojawić się gratulacje wraz z sugestią dotyczącą dalszego działania. (Tak, to nie koniec historii).

Przed opublikowaniem własnego rozwiązania polecamy zapoznanie się z rozwiązaniami innych użytkowników.

Ostatnia rzecz: niektóre zadania mają na końcu listę podpowiedzi/wskazówek. Jako że napisaliśmy już jak rozwiązać to zadanie, w sekcji wskazówek znajdziesz kilka ciekawostek o CheckiO.
