Hur fungerar det?

När du påbörjar en uppgift består koden alltid av en "tom" funktion (där du sedan ska skriva din kod) och ett antal testfall som kontrollerar att din lösning är korrekt. När du skriver din funktion är det viktigt att du returnerar resultatet och inte bara skriver ut det. {% if interpreter.slug == "js-node" %} Med andra ord behöver du använda return-kommandot istället för funktionen console.log. {% else %} Med andra ord behöver du använda return-kommandot istället för funktionen print. Förklaring {% endif %}

Testfallen efter din funktion kan användas för att kontrollera din lösning innan du skickar in den. För att göra detta, klicka på "Run Code"-knappen. När du sedan klickar på "Check Solution"-knappen kommer CheckiO att köra ytterligare testfall för att kontrollera att din lösning är korrekt.

Sist men inte minst så har vissa uppdrag en lista med ledtrådar som du kan läsa vid behov, men eftersom vi redan har berättat hur du löser den här uppgiften så är "ledtrådarna" i detta fallet några intressanta fakta om CheckiO.
