Stellen Sie sich vor, Sie müssen zwei Zahlen multiplizieren und das Ergebnis drucken (ausgeben), damit Sie es in der Konsole sehen können. Zum Beispiel: 5 und 6.

let a = 5;

Sie können die Variable a ändern, das Skript erneut ausführen und das unterschiedliche Ergebnis sehen

let b = 6;

Ausgabe:

36

Oder Sie können Ihren Code zweimal kopieren (ohne let in der Kopie), die Variablen ändern und das Ergebnis von zwei Multiplikationen sehen

let a = 5;
let b = 6;
console.log(a*b);


a = 6;
b = 6;
console.log(a*b);

Ausgabe:

30
36

Jetzt kann ich Ihnen zeigen, wie die Funktion den obigen Code vereinfachen kann.

function multTwo(a, b) {

    return a*b;
}

Ausgabe:

Ja, natürlich. Hier ist die Funktion noch einmal:

function multTwo(a, b) {

    return a*b;
}

Sie beginnt immer mit function.

Der Name der Funktion multTwo - Sie werden den Namen verwenden, um die Funktion in Zukunft aufzurufen.

Zwei Argumente (a, b) - das sind Variablen, die ihre Werte erhalten werden ihre Werte erhalten, wenn die Funktion aufgerufen wird. Wenn wir zum Beispiel die Funktion multTwo(5, 6) - innerhalb der Funktion aufrufen, setzen wir die Variable a auf 5 und die Variable b auf 6.

Sie können auch Funktionsdefinitionen mit Typ-Hinweisen sehen.

function multTwo(a: number, b: number): number {

    return a*b;
}

Lassen Sie sich nicht von : number verwirren. Sie erklärt dem Benutzer (jemandem, der den Code liest) den erwarteten Typ der Argumente und den Typ des Ergebnisses des Funktionsaufrufs. Aber es ist in Ordnung, diese Hinweise erst einmal zu ignorieren.

Jetzt sagt der Block der Funktion, der sich innerhalb von { und } und return befindet, was das Ergebnis der Funktionsaufrufe return a*b sein soll.

In unserem Beispiel console.log(multTwo(5, 6)) - wird das Ergebnis der Multiplikation von zwei Werten 5 und 6 zurückgegeben und als erstes Argument für die Funktion console.log() verwendet.

Ihre Funktion kann mehr als eine Codezeile haben. Lassen Sie uns zum Beispiel eine weitere Variable c verwenden, in der wir das Ergebnis der Multiplikation ablegen und das Ergebnis zurückgeben.

function multTwo(a, b) {

    let c = a*b;
    
    return c;
}

Oben im Code-Editor haben Sie zwei Schaltflächen, "Code ausführen" und "Lösung prüfen".

Wenn Sie die erste drücken, wird der Code aus dem Code-Editor ausgeführt. Wenn Sie auf "Lösung prüfen" klicken, prüft das System Ihre Funktion und zeigt das Feld "Glückwunsch" an, wenn sie wie erwartet funktioniert.

Wenn etwas nicht wie erwartet funktioniert, zeigt Ihnen das Panel den Aufruf der Funktion an, die den falschen Wert zurückgibt

oder wenn es sich um einen Fehler in assert handelt
