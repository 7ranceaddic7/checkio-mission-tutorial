Початковий код твого рішення буде містити "порожню" функцію (яку потрібно наповнити) та набір тестів (asserts) під цією функцією. Asserts які йдуть після оголошення функції можна використовувати для того, щоб перевірити себе використовуйте кноку Run Code. CheckiO також буде має кілька додаткових тестів для того, щоб перевірити рішення, коли ти натиснеш кнопку Check Solution.

Якщо рішення не пройшло тестування, то на правій панелі буде виведено повідомлення про помилку.

І останнє, деякі завдання наприкінці мають список підказок для вирішення. Але т.к. в цьому завданні ми вже розповіли як вирішувати, то в хінтах ми додамо кілька цікавих фактів про CheckiO
