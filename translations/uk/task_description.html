Це вступна місія, метою якої є показати, як вирішувати завдання на CheckiO та як отримати максимум від цього. Коли ця місія буде вирішена ще одна станція буде доступна, з більш складними місіями.

Це найпростіша місія. Напишіть функцію, яка отримуватиме 2 числа та повертатиме результат твору цих чисел.

Вхідні дані: Два аргумента. Оба int

Вихідні дані: Int.

Приклад: 

{{init_code_tmpl}}

Як це працьє?:

Початковий код твого рішення буде містити "порожню" функцію (яку потрібно наповнити) та набір тестів (asserts) під цією функцією. Зверніть увагу, що твоя функція повинна повертати результат, а не виводити його. {% if interpreter.slug == "js-node" %} Це означає, що потрібно використовувати команду return вместо функції console.log. {% else %} Це означає, що потрібно використовувати команду return замість фунції print. Невеличке поясненя різниці. {% endif %}

Asserts які йдуть після оголошення функції можна використовувати для того, щоб перевірити себе використовуйте кноку Run Code. CheckiO також буде має кілька додаткових тестів для того, щоб перевірити рішення, коли ти натиснеш кнопку Check Solution.

Якщо рішення не пройшло тестування, то на правій панелі буде виведено повідомлення про помилку.

Щоб розв'язати задачу “порожня” функція, повинна бути замінена на наступний код.

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Спробуй тепер натиснути Check Solution.

Якщо рішення пройде всі тести на правій панелі, має з'явитися привітання та пропозиція наступних дій. (Так, це ще не кінець історії)
